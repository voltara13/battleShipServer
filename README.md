# Сервер морского боя

### Работа велась на Qt 6.0, Cmake 3.2

Простой сервер для игры в морской бой. Не обладает каким-либо интелектом, бьёт по рандомной клетке (учитывая выбитые клетки вокруг подбитых кораблей)
